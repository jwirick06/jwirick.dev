# Jonathan Wirick - Personal Portfolio

![Live Demo](https://img.shields.io/badge/demo-online-green.svg)

A personal portfolio website to tell you a bit about me, showcase my projects, resume, and technical skills.

**Live Site:** [https://jwirick.dev](https://jwirick.dev)

## Tech Stack

* **Framework:** Astro
* **Styling:** Tailwind CSS
* **Deployment:** GitHub Pages

## Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
* **Project Showcase:** Dynamic grid layout for recent work.
* **Automated Deployment:** Handled via GitHub Actions.

## Running Locally

To run this project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jwirick06/jwirick.dev
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
